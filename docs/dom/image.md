图片的后缀格式（扩展名）主要分为**传统格式**、**现代高效格式**、**动态图片格式**和**专业/特殊格式**。以下是常见的分类及说明：

### 1. 传统常见格式（兼容性最好）
-   **.jpg / .jpeg**：最流行的有损压缩格式，适合照片、复杂图像，文件较小但会损失部分细节。
-   **.png**：无损压缩，支持透明背景，适合图标、截图、需要清晰边缘的图形。
-   **.gif**：支持动态图（动画）和简单透明，但颜色只有256种，画质一般。
-   **.bmp**：位图格式，无压缩，画质极高但文件非常大，现已少用。

### 2. 现代高效格式（性能更优）
-   **.webp**：谷歌推出，同时支持有损/无损/透明/动画，文件比JPEG和PNG更小，现代网站普遍使用。
-   **.heic**：苹果iOS设备（iPhone/iPad）默认照片格式，压缩率高、画质好，但Windows和部分软件需解码才能打开。
-   **.heif**：HEIC的通用版本，支持多张图像和连拍。
-   **.avif**：基于AV1编码的新格式，压缩率比WebP更高，支持HDR和广色域，正在逐步普及。

### 3. 动态图片格式
-   **.gif**（见上文）
-   **.webp**（也支持动画）
-   **.apng**：支持高色深的动态PNG，画质优于GIF但兼容性稍差。

### 4. 专业/矢量格式（无损缩放，适合设计）
-   **.svg**：矢量图，无限放大不模糊，常用于网页图标、LOGO（用代码编写）。
-   **.eps**：印刷出版常用矢量格式，支持PostScript。
-   **.ai**：Adobe Illustrator源文件。
-   **.cdr**：CorelDRAW源文件。

### 5. 特殊/RAW格式（相机原始底片）
这些是专业相机拍摄的未经处理的原始数据，文件巨大，需要专门软件查看：
-   **.nef**（尼康）
-   **.cr2 / .cr3**（佳能）
-   **.arw**（索尼）
-   **.dng**（Adobe的通用RAW格式）
-   **.raf**（富士）

### 总结建议：
-   **发朋友圈/网页**：首选 `.jpg` 或 `.webp`（小、清晰）。
-   **需要透明背景**：用 `.png`。
-   **做动图**：用 `.gif` 或 `.webp`。
-   **设计稿/LOGO**：用 `.svg`（无限放大）。
-   **iPhone传电脑**：注意 `.heic` 格式可能需要转换。
