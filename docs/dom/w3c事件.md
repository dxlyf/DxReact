## 1. 鼠标事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `mousedown` | 是 | 是 | 阻止开始文本选择/拖拽/选中UI控件（如按钮按下样式） | 按下鼠标按钮 |
| `mouseup` | 是 | 是 | 阻止释放后的交互（如链接跳转） | 释放鼠标按钮 |
| `click` | 是 | 是 | 阻止链接跳转、按钮提交、复选框/单选框状态改变 | 单击鼠标 |
| `dblclick` | 是 | 是 | 阻止双击选择文本（通常由系统处理，部分可取消） | 双击鼠标 |
| `mouseover` | 是 | 是 | 阻止默认的 tooltip 显示（如 title 属性） | 鼠标移入元素（含子元素） |
| `mouseout` | 是 | 是 | 同上 | 鼠标移出元素（含子元素） |
| `mouseenter` | 否 | 是 | 同上 | 首次进入元素边界（不冒泡） |
| `mouseleave` | 否 | 是 | 同上 | 离开元素边界（不冒泡） |
| `mousemove` | 是 | 否 | 无法阻止 | 鼠标在元素内移动 |
| `contextmenu` | 是 | 是 | **阻止浏览器右键菜单弹出** | 右键点击或按菜单键 |

## 2. 键盘事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `keydown` | 是 | 是 | 阻止输入字符、阻止快捷键（F5/Ctrl+S等）、阻止退格/删除触发前进后退 | 按下任意键 |
| `keypress` | 是 | 是 | 同上（但仅针对字符键，已废弃） | 按下字符键 |
| `keyup` | 是 | 是 | 阻止释放键后的默认行为（极少需要） | 释放按键 |

---

## 3. 焦点事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `focus` | 否 | 否 | 不可取消 | 元素获得焦点 |
| `blur` | 否 | 否 | 不可取消 | 元素失去焦点 |
| `focusin` | 是 | 否 | 不可取消 | 即将获得焦点（冒泡版 focus） |
| `focusout` | 是 | 否 | 不可取消 | 即将失去焦点（冒泡版 blur） |

> ⚠️ 焦点事件无法取消，但可以调用 `preventDefault()` 在 `mousedown` 中阻止焦点转移。

## 4. 表单事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `submit` | 是 | 是 | **阻止表单提交**（不发送请求） | 表单提交时 |
| `change` | 是 | 否 | 无法阻止 | 值改变并失去焦点 |
| `input` | 是 | 否 | 无法阻止 | 值立即变化 |
| `reset` | 是 | 否 | 无法阻止 | 表单重置 |
| `select` | 是 | 否 | 无法阻止 | 选中文本框中的文本 |

## 5. 视图/资源事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `load` | 否 | 否 | 不可取消 | 页面/资源加载完成 |
| `beforeunload` | 否 | 是 | **弹出确认对话框，阻止页面关闭/刷新** | 页面即将卸载 |
| `unload` | 否 | 否 | 不可取消（已不推荐使用） | 页面正在卸载 |
| `resize` | 是 | 否 | 不可取消 | 窗口大小改变 |
| `scroll` | 是 | 否 | 不可取消（但可通过 CSS 或 JS 阻止滚动行为本身） | 滚动条滚动 |
| `error` | 是 | 否 | 不可取消 | 资源加载失败或脚本错误 |

## 6. 触摸与指针事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `touchstart` | 是 | 是 | **阻止滚动、缩放、长按菜单、页面拖动** | 手指触摸屏幕 |
| `touchmove` | 是 | 是 | **阻止滚动**（经典用法） | 手指在屏幕上移动 |
| `touchend` | 是 | 是 | 阻止某些触屏默认动作（如恢复滚动惯性） | 手指离开屏幕 |
| `touchcancel` | 是 | 否 | 不可取消 | 系统中断触摸（如来电） |
| `pointerdown` | 是 | 是 | 同 touchstart + 鼠标/笔的默认行为 | 指针接触屏幕 |
| `pointerup` | 是 | 是 | 同上 | 指针离开屏幕 |
| `pointermove` | 是 | 是 | 阻止指针移动触发的默认行为 | 指针移动 |
| `pointerover` | 是 | 是 | 阻止系统样式变化（如 hover） | 指针移入元素 |
| `pointerout` | 是 | 是 | 同上 | 指针移出元素 |
| `pointerenter` | 否 | 是 | 同上 | 首次进入元素边界 |
| `pointerleave` | 否 | 是 | 同上 | 离开元素边界 |

## 8. 拖拽事件

| 事件类型 | 是否冒泡 | 是否可取消 | 可阻止的默认行为 | 触发时机 |
| :--- | :--- | :--- | :--- | :--- |
| `dragstart` | 是 | 是 | **阻止拖拽开始** | 开始拖拽元素 |
| `drag` | 是 | 是 | 阻止拖拽光标变化等 | 拖拽过程中 |
| `dragend` | 是 | 是 | 阻止拖拽结束后恢复默认状态 | 拖拽结束 |
| `dragenter` | 是 | 是 | 阻止默认的拖拽反馈（如不显示可放下光标） | 拖拽元素进入目标 |
| `dragover` | 是 | 是 | **必须取消默认行为才能成为有效 drop 目标** | 拖拽在目标上移动 |
| `dragleave` | 是 | 是 | 同上 | 拖拽离开目标 |
| `drop` | 是 | 是 | **阻止文件/数据被放下** | 释放拖拽元素 |

> 📌 在 `dragover` 中调用 `preventDefault()` 是让元素允许被 drop 的必要操作。
