创建一个可排序、可筛选的数据表格组件，要求：
1. 功能描述：显示数据表格，支持排序、筛选、自定义列
2. 输入参数：
   - data：Array，[]，表格数据
   - columns：Array，[]，列配置
   - loading：Boolean，false，加载状态
   - pagination：Object，{ pageSize: 10 }，分页配置
3. 输出事件：
   - sort：{ column, order }，排序时触发
   - filter：{ filters }，筛选时触发
   - row-click：{ row, index }，点击行时触发
4. 插槽：
   - header：表格头部自定义
   - column-[field]：列内容自定义
   - actions：操作列内容
5. 样式要求：使用Tailwind CSS
6. 响应式：小屏幕时水平滚动



创建一个带验证的表单输入组件，要求：
1. 组件名称：ValidatedInput
2. 功能：输入框+实时验证+错误提示
3. 输入参数：
   - type：String，'text'，输入类型
   - modelValue：String，''，双向绑定值
   - rules：Array，[]，验证规则数组
   - label：String，''，标签文本
   - placeholder：String，''，占位符
4. 输出事件：
   - update:modelValue：String，值变化时触发
   - validate：Boolean，验证状态变化时触发
5. 插槽：
   - prefix：输入框前缀图标
   - suffix：输入框后缀图标
6. 样式：错误状态红色边框，正确状态绿色边框
