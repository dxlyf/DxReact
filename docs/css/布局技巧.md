## CSS 布局技巧大全

除了 Flex 和 Grid，还有很多实用的布局技巧。我来按类别整理：

---

## 一、经典布局技巧

### 1. 圣杯布局 / 双飞翼布局
**三栏布局，中间自适应，两边固定**

```css
/* 现代方案：Grid */
.holy-grail {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  grid-template-areas: "left main right";
  min-height: 100vh;
}

.left { grid-area: left; }
.main { grid-area: main; }
.right { grid-area: right; }

/* 响应式 */
@media (max-width: 768px) {
  .holy-grail {
    grid-template-columns: 1fr;
    grid-template-areas: 
      "left"
      "main"
      "right";
  }
}
```

### 2. 瀑布流布局
**像花瓣网、Pinterest 那样的错落布局**

```css
/* CSS 方案（简单版） */
.waterfall {
  columns: 3;  /* 列数 */
  column-gap: 20px;
}

.waterfall-item {
  break-inside: avoid;  /* 防止内容断裂 */
  margin-bottom: 20px;
}

/* 更精确的 Grid 方案 */
.waterfall-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

.waterfall-grid-item {
  grid-row: span 1;  /* 实际需要用 JS 计算跨度 */
}
```

### 3. 粘性布局
**元素在滚动到特定位置时固定**

```css
/* 粘性头部 */
.sticky-header {
  position: sticky;
  top: 0;
  z-index: 100;
  background: white;
}

/* 粘性侧边栏 */
.sticky-sidebar {
  position: sticky;
  top: 20px;
  height: fit-content;
}

/* 表格表头粘性 */
th {
  position: sticky;
  top: 0;
  background: white;
  z-index: 10;
}
```

## 二、元素定位技巧

### 4. 绝对定位居中
```css
/* 方法1：transform 法 */
.center-absolute {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 方法2：margin:auto 法（需要明确宽高） */
.center-margin {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  margin: auto;
  width: 200px;
  height: 100px;
}

/* 方法3：inset 法（现代浏览器） */
.center-inset {
  position: absolute;
  inset: 0;
  margin: auto;
  width: 200px;
  height: 100px;
}
```

### 5. 宽高比控制
```css
/* 16:9 视频容器 */
.aspect-16-9 {
  position: relative;
  width: 100%;
  padding-top: 56.25%; /* 9/16 = 0.5625 */
}

.aspect-16-9 > * {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}

/* 现代方案：aspect-ratio */
.aspect-modern {
  aspect-ratio: 16 / 9;
  width: 100%;
}

/* 正方形 */
.square {
  aspect-ratio: 1 / 1;
}

/* 4:3 老电视比例 */
.aspect-4-3 {
  aspect-ratio: 4 / 3;
}
```

### 6. 等高列布局
```css
/* Flex 方案（自动等高） */
.equal-height-flex {
  display: flex;
}

.equal-height-flex > * {
  flex: 1;
}

/* Grid 方案 */
.equal-height-grid {
  display: grid;
  grid-auto-rows: 1fr;  /* 所有行等高 */
  gap: 20px;
}

/* 表格方案（兼容性最好） */
.equal-height-table {
  display: table;
  width: 100%;
}

.equal-height-table > * {
  display: table-cell;
}
```

## 三、文本布局技巧

### 7. 文本截断与省略
```css
/* 单行文本省略 */
.truncate {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

/* 多行文本省略 */
.truncate-2 {
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
  overflow: hidden;
}

/* 渐变消失效果 */
.fade-out {
  position: relative;
  max-height: 100px;
  overflow: hidden;
}

.fade-out::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  height: 40px;
  background: linear-gradient(transparent, white);
}
```

### 8. 文字环绕
```css
/* 图片左环绕 */
.float-left {
  float: left;
  margin-right: 20px;
  margin-bottom: 10px;
}

/* 形状环绕 */
.shape-outside {
  float: left;
  shape-outside: circle(50%);  /* 圆形环绕 */
  clip-path: circle(50%);
  margin-right: 20px;
}

/* 菱形环绕 */
.shape-polygon {
  float: left;
  shape-outside: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
  clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
}
```

## 四、响应式技巧

### 9. 容器查询（新特性）
```css
/* 基于父容器大小响应，而非视口 */
.card-container {
  container-type: inline-size;
  container-name: card;
}

@container card (max-width: 400px) {
  .card {
    flex-direction: column;
  }
  
  .card-image {
    width: 100%;
  }
}

/* 结合样式查询 */
@container style(--theme: dark) {
  .card {
    background: #333;
    color: white;
  }
}
```

### 10. 钳位函数
```css
/* 流体排版 */
.fluid-text {
  font-size: clamp(1rem, 3vw, 2rem);
}

/* 流体间距 */
.fluid-padding {
  padding: clamp(1rem, 5%, 3rem);
}

/* 流体宽度 */
.fluid-width {
  width: clamp(300px, 80%, 1200px);
  margin: 0 auto;
}

/* 动态网格 */
.dynamic-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(clamp(200px, 30%, 300px), 1fr));
}
```

## 五、创意布局技巧

### 11. 毛玻璃效果
```css
.glass {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.1);
}

/* 多重玻璃 */
.glass-card {
  position: relative;
  background: white;
}

.glass-card::before {
  content: '';
  position: absolute;
  inset: 0;
  background: inherit;
  backdrop-filter: blur(10px);
  z-index: -1;
}
```

### 12. 边框装饰
```css
/* 渐变边框 */
.gradient-border {
  border: 4px solid transparent;
  border-image: linear-gradient(45deg, #f06, #9f6) 1;
  border-radius: 8px;  /* 注意：border-image 不支持圆角 */
}

/* 圆角渐变边框（使用伪元素） */
.radius-gradient-border {
  position: relative;
  background: white;
  border-radius: 8px;
}

.radius-gradient-border::before {
  content: '';
  position: absolute;
  inset: -2px;
  background: linear-gradient(45deg, #f06, #9f6);
  border-radius: 10px;
  z-index: -1;
}

/* 虚线边框动画 */
.dashed-animation {
  border: 2px dashed #333;
  animation: dash 20s linear infinite;
}

@keyframes dash {
  to {
    border-image: repeating-linear-gradient(45deg, #333 0, #333 10px, transparent 10px, transparent 20px) 1;
  }
}
```

### 13. 分离轴滚动
```css
/* 水平滚动卡片 */
.horizontal-scroll {
  display: flex;
  overflow-x: auto;
  gap: 20px;
  padding: 20px;
  scroll-snap-type: x mandatory;
}

.horizontal-scroll > * {
  flex: 0 0 300px;
  scroll-snap-align: start;
}

/* 垂直滚动 + 水平滚动组合 */
.mixed-scroll {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  overflow-y: auto;
  max-height: 500px;
  gap: 20px;
}

/* 隐藏滚动条但保留功能 */
.hide-scrollbar {
  scrollbar-width: none;  /* Firefox */
  -ms-overflow-style: none;  /* IE */
}

.hide-scrollbar::-webkit-scrollbar {
  display: none;  /* Chrome/Safari */
}
```

## 六、实用小技巧

### 14. 长单词换行
```css
.word-break {
  overflow-wrap: break-word;  /* 长单词折行 */
  word-wrap: break-word;      /* 兼容写法 */
  word-break: break-all;      /* 更激进的折行 */
  hyphens: auto;              /* 添加连字符 */
}
```

### 15. 平滑滚动
```css
html {
  scroll-behavior: smooth;  /* 页面内锚点平滑滚动 */
}

/* 特定容器平滑滚动 */
.smooth-container {
  scroll-behavior: smooth;
  overscroll-behavior: contain;  /* 阻止滚动链 */
}
```

### 16. 打印样式优化
```css
@media print {
  /* 隐藏不需要打印的元素 */
  .no-print {
    display: none !important;
  }
  
  /* 确保链接显示完整URL */
  a[href]::after {
    content: " (" attr(href) ")";
  }
  
  /* 分页控制 */
  .page-break {
    page-break-before: always;
  }
  
  /* 避免内部断页 */
  .keep-together {
    page-break-inside: avoid;
  }
}
```

## 七、现代 CSS 新特性

### 17. 级联层
```css
@layer reset, base, components, utilities;

@layer reset {
  * {
    margin: 0;
    padding: 0;
  }
}

@layer components {
  .card {
    background: white;
    border-radius: 8px;
  }
}

@layer utilities {
  .mt-auto {
    margin-top: auto;
  }
}
```

### 18. 嵌套语法
```css
.card {
  padding: 1rem;
  
  /* 直接嵌套 */
  & .title {
    font-size: 1.25rem;
  }
  
  /* 嵌套伪类 */
  &:hover {
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
  }
  
  /* 嵌套媒体查询 */
  @media (max-width: 768px) {
    padding: 0.5rem;
  }
}
```

### 19. 自定义属性
```css
:root {
  --primary: #007bff;
  --spacing: 1rem;
  --container-width: min(1200px, 90%);
}

.container {
  width: var(--container-width);
  margin: 0 auto;
  padding: calc(var(--spacing) * 2);
}

.button {
  background: var(--primary);
  margin: var(--spacing);
}

/* 动态修改 */
@media (prefers-color-scheme: dark) {
  :root {
    --primary: #0d6efd;
  }
}
```

## 总结

这些技巧覆盖了：
- **经典布局**：圣杯、瀑布流、粘性布局
- **元素定位**：居中、宽高比、等高列
- **文本处理**：截断、环绕
- **响应式**：容器查询、钳位函数
- **创意效果**：毛玻璃、渐变边框
- **现代特性**：级联层、嵌套、自定义属性

掌握这些技巧，基本可以应对 99% 的布局需求。关键是**在合适的场景用合适的方法**。
