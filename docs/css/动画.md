对于树节点（tree node）的展开和收缩动画，我推荐使用**高度过渡动画**，这是最自然、用户体验最好的方式。下面我给你提供几种不同复杂度的实现方案。

## 方案一：CSS 简单高度过渡（最常用）

```vue
<template>
  <div class="tree-node">
    <div class="node-title" @click="toggle">
      <span>{{ title }}</span>
      <span class="arrow" :class="{ expanded: isExpanded }">▶</span>
    </div>
    
    <!-- 子节点容器 -->
    <div class="children-container" :class="{ expanded: isExpanded }">
      <div class="children-wrapper">
        <slot></slot>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'

defineProps({
  title: String
})

const isExpanded = ref(false)

const toggle = () => {
  isExpanded.value = !isExpanded.value
}
</script>

<style scoped>
.tree-node {
  margin-left: 20px;
}

.node-title {
  cursor: pointer;
  padding: 8px;
  display: flex;
  align-items: center;
  gap: 8px;
  user-select: none;
}

.node-title:hover {
  background-color: #f5f5f5;
}

.arrow {
  display: inline-block;
  transition: transform 0.3s ease;
  font-size: 12px;
}

.arrow.expanded {
  transform: rotate(90deg);
}

/* 核心动画部分 */
.children-container {
  display: grid;
  grid-template-rows: 0fr;
  transition: grid-template-rows 0.3s ease-out;
  overflow: hidden;
}

.children-container.expanded {
  grid-template-rows: 1fr;
}

.children-wrapper {
  min-height: 0; /* 必须设置，否则grid动画不生效 */
}
</style>
```

**优点**：纯CSS实现，性能好，代码简单  
**缺点**：需要知道具体高度（这里用grid技巧解决了）

## 方案二：使用 Vue Transition + 动态高度（最灵活）

```vue
<template>
  <div class="tree-node">
    <div class="node-title" @click="toggle">
      <span>{{ title }}</span>
      <span class="arrow" :class="{ expanded: isExpanded }">▶</span>
    </div>
    
    <!-- 使用 Vue Transition -->
    <transition
      @before-enter="beforeEnter"
      @enter="enter"
      @before-leave="beforeLeave"
      @leave="leave"
    >
      <div v-show="isExpanded" class="children-container" ref="containerRef">
        <div class="children-wrapper">
          <slot></slot>
        </div>
      </div>
    </transition>
  </div>
</template>

<script setup>
import { ref } from 'vue'

defineProps({
  title: String
})

const isExpanded = ref(false)
const containerRef = ref(null)

const toggle = () => {
  isExpanded.value = !isExpanded.value
}

// 展开动画
const beforeEnter = (el) => {
  el.style.height = '0'
  el.style.opacity = '0'
}

const enter = (el) => {
  el.style.height = el.scrollHeight + 'px'
  el.style.opacity = '1'
}

// 收缩动画
const beforeLeave = (el) => {
  el.style.height = el.scrollHeight + 'px'
  el.style.opacity = '1'
}

const leave = (el) => {
  el.style.height = '0'
  el.style.opacity = '0'
}
</script>

<style scoped>
.tree-node {
  margin-left: 20px;
}

.node-title {
  cursor: pointer;
  padding: 8px;
  display: flex;
  align-items: center;
  gap: 8px;
  user-select: none;
}

.node-title:hover {
  background-color: #f5f5f5;
}

.arrow {
  display: inline-block;
  transition: transform 0.3s ease;
  font-size: 12px;
}

.arrow.expanded {
  transform: rotate(90deg);
}

/* 动画容器 */
.children-container {
  overflow: hidden;
  transition: all 0.3s ease;
}
</style>
```

**优点**：完美适应任何高度，动画流畅  
**缺点**：需要JavaScript配合

## 方案三：Element Plus 风格的优雅动画（推荐）

这是模仿Element Plus树组件的动画效果，非常优雅：

```vue
<template>
  <div class="tree-node">
    <div 
      class="tree-node__content" 
      @click="toggle"
      :class="{ 'is-expanded': isExpanded }"
    >
      <span class="tree-node__expand-icon" :class="{ expanded: isExpanded }">
        <svg viewBox="0 0 1024 1024" width="12" height="12">
          <path d="M304.4 835.8L679.6 512 304.4 188.2v647.6z" fill="currentColor"/>
        </svg>
      </span>
      <span class="tree-node__label">{{ title }}</span>
    </div>
    
    <transition
      @before-enter="beforeEnter"
      @enter="enter"
      @after-enter="afterEnter"
      @before-leave="beforeLeave"
      @leave="leave"
      @after-leave="afterLeave"
    >
      <div v-show="isExpanded" class="tree-node__children" ref="childrenRef">
        <slot></slot>
      </div>
    </transition>
  </div>
</template>

<script setup>
import { ref, nextTick } from 'vue'

const props = defineProps({
  title: String,
  defaultExpanded: Boolean
})

const isExpanded = ref(props.defaultExpanded || false)
const childrenRef = ref(null)

const toggle = () => {
  isExpanded.value = !isExpanded.value
}

// 展开动画
const beforeEnter = (el) => {
  el.style.height = '0'
  el.style.opacity = '0'
  el.style.marginTop = '0'
  el.style.marginBottom = '0'
}

const enter = (el) => {
  el.style.height = el.scrollHeight + 'px'
  el.style.opacity = '1'
  el.style.marginTop = ''
  el.style.marginBottom = ''
}

const afterEnter = (el) => {
  el.style.height = ''
}

// 收缩动画
const beforeLeave = (el) => {
  el.style.height = el.scrollHeight + 'px'
  el.style.opacity = '1'
}

const leave = (el) => {
  el.style.height = '0'
  el.style.opacity = '0'
  el.style.marginTop = '0'
  el.style.marginBottom = '0'
}

const afterLeave = (el) => {
  el.style.height = ''
  el.style.marginTop = ''
  el.style.marginBottom = ''
}
</script>

<style scoped>
.tree-node {
  line-height: 26px;
  font-size: 14px;
}

.tree-node__content {
  display: flex;
  align-items: center;
  cursor: pointer;
  padding: 4px 0;
  transition: background-color 0.2s ease;
  border-radius: 4px;
}

.tree-node__content:hover {
  background-color: #f5f7fa;
}

.tree-node__expand-icon {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  width: 24px;
  height: 24px;
  color: #97a8be;
  transition: transform 0.2s ease;
  transform: rotate(0deg);
}

.tree-node__expand-icon.expanded {
  transform: rotate(90deg);
}

.tree-node__label {
  margin-left: 4px;
  color: #606266;
}

.tree-node__children {
  overflow: hidden;
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
  margin-left: 24px;
}

/* 嵌套样式 */
.tree-node .tree-node {
  margin-left: 24px;
}
</style>
```

## 方案四：使用 GSAP 实现更流畅的动画（专业级）

如果追求极致的动画效果，可以使用 GSAP：

```vue
<template>
  <div class="tree-node">
    <div class="node-title" @click="toggle">
      <span>{{ title }}</span>
      <span class="arrow" :class="{ expanded: isExpanded }">▶</span>
    </div>
    
    <div class="children-container" ref="childrenRef" v-show="isExpanded">
      <slot></slot>
    </div>
  </div>
</template>

<script setup>
import { ref, watch } from 'vue'
import gsap from 'gsap'

const props = defineProps({
  title: String
})

const isExpanded = ref(false)
const childrenRef = ref(null)

const toggle = () => {
  isExpanded.value = !isExpanded.value
}

watch(isExpanded, (newVal) => {
  if (newVal) {
    // 展开动画
    gsap.fromTo(childrenRef.value,
      { height: 0, opacity: 0 },
      { 
        height: 'auto', 
        opacity: 1,
        duration: 0.3,
        ease: 'power2.out'
      }
    )
  } else {
    // 收缩动画
    gsap.to(childrenRef.value, {
      height: 0,
      opacity: 0,
      duration: 0.3,
      ease: 'power2.in'
    })
  }
}, { immediate: true })
</script>

<style scoped>
.tree-node {
  margin-left: 20px;
}

.node-title {
  cursor: pointer;
  padding: 8px;
  display: flex;
  align-items: center;
  gap: 8px;
  user-select: none;
}

.node-title:hover {
  background-color: #f5f5f5;
}

.arrow {
  display: inline-block;
  transition: transform 0.3s ease;
  font-size: 12px;
}

.arrow.expanded {
  transform: rotate(90deg);
}

.children-container {
  overflow: hidden;
}
</style>
```

## 动画效果对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| CSS Grid | 纯CSS实现，性能好 | 浏览器兼容性稍差 | 现代项目，简单树结构 |
| Vue Transition | 适应任意高度，兼容性好 | 需要JS计算 | 大部分项目，推荐使用 |
| Element Plus风格 | 视觉效果专业，有弹性感 | 代码稍复杂 | 企业级后台，追求体验 |
| GSAP | 动画最流畅，效果丰富 | 需要额外引入库 | 对动画要求极高的场景 |

## 动画贝塞尔曲线推荐

对于树节点动画，推荐以下几种缓动函数：

```css
/* 标准缓动 - 推荐 */
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);

/* 弹性感强一些 */
transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);

/* 轻快自然 */
transition: all 0.25s cubic-bezier(0.25, 0.46, 0.45, 0.94);

/* 稍微有弹性的感觉 */
transition: all 0.3s cubic-bezier(0.68, -0.55, 0.265, 1.55);
```
