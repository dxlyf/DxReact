# HTML 元素标签完整列表

以下是所有 HTML 元素的完整分类列表，包括标准 HTML 元素和 SVG 元素。

## 📄 标准 HTML 元素

### 文档结构元素
| 元素 | 描述 | 示例 |
|------|------|------|
| `<!DOCTYPE>` | 文档类型声明 | `<!DOCTYPE html>` |
| `<html>` | HTML 文档根元素 | `<html lang="en">` |
| `<head>` | 文档头部 | `<head><title>Page</title></head>` |
| `<title>` | 文档标题 | `<title>My Page</title>` |
| `<body>` | 文档主体 | `<body>内容</body>` |
| `<meta>` | 元数据 | `<meta charset="UTF-8">` |
| `<link>` | 外部资源链接 | `<link rel="stylesheet" href="style.css">` |
| `<style>` | 内联样式 | `<style>body { color: red; }</style>` |
| `<script>` | 脚本代码 | `<script>console.log('Hello')</script>` |
| `<base>` | 基础 URL | `<base href="https://example.com/">` |
| `<noscript>` | 脚本未支持时内容 | `<noscript>请启用 JavaScript</noscript>` |

## 💡 使用提示

### HTML5 新增元素
```html
<!-- 语义化元素 -->
<header>, <footer>, <main>, <section>, <article>, <aside>, <nav>

<!-- 功能增强 -->
<details>, <summary>, <dialog>, <progress>, <meter>

<!-- 媒体元素 -->
<audio>, <video>, <source>, <track>
```

### 已废弃元素（避免使用）
```html
<!-- 已废弃的元素 -->
<acronym>, <applet>, <basefont>, <big>, <center>
<dir>, <font>, <frame>, <frameset>, <noframes>
<strike>, <tt>
```

### SVG 使用示例
```html
<svg width="200" height="200" viewBox="0 0 100 100">
  <defs>
    <linearGradient id="grad1" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="red"/>
      <stop offset="100%" stop-color="blue"/>
    </linearGradient>
  </defs>
  
  <rect x="10" y="10" width="80" height="80" fill="url(#grad1)"/>
  <circle cx="50" cy="50" r="30" fill="green" opacity="0.7"/>
  <text x="50" y="50" text-anchor="middle" fill="white">SVG</text>
</svg>
```

这个列表包含了绝大多数常用的 HTML 和 SVG 元素，可以作为开发时的参考手册使用。
