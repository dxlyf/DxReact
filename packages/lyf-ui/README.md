# Ant Design 完整组件列表（基于 Ant Design 5.0）

根据 Ant Design 官方文档和组件总览，Ant Design 5.0 提供 **8 大类别，共计 70+ 个高质量 React 组件** 。以下是完整的组件分类列表：


## 一、通用组件（General）

| 组件 | 描述 |
|:---|:---|
| **Button** | 按钮，支持多种类型、尺寸和状态 |
| **Icon** | 图标，内置图标库 |
| **Typography** | 排版，包括标题、文本、段落 |
| **Divider** | 分割线 |
| **Space** | 间距，设置子元素间距 |
| **Flex** | 弹性布局，封装 CSS Flex |
| **Grid** | 栅格，24栅格系统 |


## 二、布局组件（Layout）

| 组件 | 描述 |
|:---|:---|
| **Layout** | 布局容器，包含 Header/Sider/Content/Footer |
| **Splitter** | 分割面板，可拖拽调整大小 |
| **Watermark** | 水印 |


## 三、导航组件（Navigation）

| 组件 | 描述 |
|:---|:---|
| **Menu** | 导航菜单，支持垂直/水平、内嵌/弹出 |
| **Dropdown** | 下拉菜单 |
| **Breadcrumb** | 面包屑 |
| **Pagination** | 分页 |
| **Steps** | 步骤条 |
| **Tabs** | 标签页 |
| **Affix** | 固钉，将元素固定在可视区域 |
| **Anchor** | 锚点，页面内导航 |


## 四、数据录入组件（Data Entry）

| 组件 | 描述 |
|:---|:---|
| **Input** | 输入框 |
| **InputNumber** | 数字输入框 |
| **AutoComplete** | 自动完成 |
| **Select** | 选择器 |
| **TreeSelect** | 树选择 |
| **Cascader** | 级联选择 |
| **Checkbox** | 复选框 |
| **Radio** | 单选框 |
| **Switch** | 开关 |
| **Slider** | 滑动输入条 |
| **Rate** | 评分 |
| **DatePicker** | 日期选择器 |
| **TimePicker** | 时间选择器 |
| **Upload** | 上传 |
| **Transfer** | 穿梭框 |
| **Mentions** | 提及 |
| **ColorPicker** | 颜色选择器 |
| **Form** | 表单（核心组件，支持 useForm 管理） |


## 五、数据展示组件（Data Display）

| 组件 | 描述 |
|:---|:---|
| **Table** | 表格（支持排序、筛选、分页、树形数据、虚拟滚动等） |
| **Tree** | 树形控件 |
| **Collapse** | 折叠面板 |
| **Carousel** | 走马灯 |
| **Card** | 卡片 |
| **List** | 列表 |
| **Badge** | 徽标 |
| **Avatar** | 头像 |
| **Calendar** | 日历 |
| **Image** | 图片（支持预览） |
| **Tag** | 标签 |
| **Descriptions** | 描述列表 |
| **Statistic** | 统计数值 |
| **QRCode** | 二维码 |
| **Masonry** | 瀑布流布局 |


## 六、反馈组件（Feedback）

| 组件 | 描述 |
|:---|:---|
| **Alert** | 警告提示 |
| **Message** | 全局提示 |
| **Notification** | 通知提醒框 |
| **Modal** | 模态框 |
| **Drawer** | 抽屉 |
| **Popconfirm** | 气泡确认框 |
| **Progress** | 进度条 |
| **Skeleton** | 骨架屏 |
| **Spin** | 加载中 |
| **Result** | 结果页 |
| **Empty** | 空状态 |


## 七、弹出层组件（Popover）

| 组件 | 描述 |
|:---|:---|
| **Tooltip** | 文字提示 |
| **Popover** | 气泡卡片 |
| **Tour** | 漫游式引导 |


## 八、其他组件（Other）

| 组件 | 描述 |
|:---|:---|
| **ConfigProvider** | 全局配置 |
| **App** | 包裹组件，提供静态方法 |
| **FloatButton** | 悬浮按钮 |
| **Segmented** | 分段控制器 |
